
#### 前言
基于 [慕课网](https://www.imooc.com/) Jokcy老师的 [Vue核心技术实战精讲](https://coding.imooc.com/class/196.html) 视频教程编写。

#### 使用须知
- **使用&参考此源码，需要您对HTML、CSS、JavaScript、webpack、Node.js等有一定的了解和实践。**

#### 使用步骤

##### 安装依赖
- 安装NPM依赖：`npm install`


##### 开发环境
- 启动项目：`npm run dev`
- 浏览器预览：__localhost:8001__


##### 生产环境
- 启动项目打包：`npm run build`
